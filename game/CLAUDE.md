# CLAUDE.md

このファイルは、リポジトリで作業する Claude Code (claude.ai/code) へのガイダンスを提供します。

## 概要

ビルドステップ・依存関係・テスト基盤なし。各ゲームは単一の自己完結したHTMLファイルとして実装されています。

- `game/breakout.html` — ブロック崩し：パドル・マルチボール・アイテム・パーティクル・Web Audio ビープ音
- `game/index.html` — 落ちモノパズル：テトリス風、ゴーストピース・壁蹴り・レベル進行あり

## 実行方法

どちらのファイルもブラウザで直接開けばOK（サーバー不要）。ライブリロードで開発する場合：

```bash
npx serve game/
# http://localhost:3000/breakout.html または /index.html を開く
```

## アーキテクチャ

両ゲームとも同じ構成パターンに従っています。単一の `<script>` ブロック内に：

1. **定数** — ボード寸法・速度・カラーテーブル
2. **状態変数** — モジュールスコープで `let` 宣言、`startGame()` でリセット
3. **更新関数** — 純粋なロジック、毎フレーム呼び出し
4. **描画関数** — Canvas 2D API、更新後に毎フレーム呼び出し
5. **ゲームループ** — `requestAnimationFrame` ベース、`dt`（デルタタイム、ms単位）を `update()` に渡す
6. **イベントリスナー** — キーボードとマウスをファイル末尾で登録

モジュールシステム・バンドラー・フレームワークは一切なし。すべてのコードはスクリプトタグ内のグローバルスコープです。
